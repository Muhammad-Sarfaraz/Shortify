# Shortify
Streamline URL shortening on the scalable NestJS framework.

## Installation

```bash
$ npm install
```
```bash
cp .env.example .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
